# Email-collectop-2.0
Сборщик писем на gulp + mj.

**Pug > Mjml > Html email creator**

### Install
```bash
$ npm i -E
```

### Run server

```bash
$ npm run start
```

### Available tasks:

#### Compile email

```bash
$ npm run build
```

#### Watch

```bash
$ npm run watch
```

Launch http://localhost:4000

##### Source pug files into:
source/views/templates

##### Build folder:
build/templates
